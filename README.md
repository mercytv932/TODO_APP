# Todo App

A professional todo application built with vanilla JavaScript.

## Features

- Add, delete, and complete todos
- Filter todos by All, Active, and Completed
- Persistent storage with localStorage
- Live stats showing total, active, and completed counts
- Input sanitization

## Technologies

- HTML
- CSS
- Vanilla JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding todos!

- DOM manipulation
- localStorage with JSON
- State management
- Event delegation
